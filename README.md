# paper-manager

Tag-based paper manager to archive papers that you have read.

## Directory structure
I hope that the single, unique, py file handles everything.

### Dir: database
This directory contains a sqlite database file and the backup files for the database.

### Dir: manager
All other py files will held in here.

### Dir: workspace
The temp directory for retrieving/storing files.

## How to run

1. (prerequisite) Install python
1. Install requirements.
1. run manage.py
